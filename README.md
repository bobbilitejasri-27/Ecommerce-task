# Simple E-commerce Website - Task 1

## Project Description
Basic e-commerce website with product listing, add to cart functionality, and cart page. Built using HTML, CSS, and JavaScript with localStorage for data storage.

## Features Completed in Task 1
- **Product Listing**: Displays products with name, price, and image
- **Add to Cart**: Users can add products to cart from homepage
- **Cart Page**: View all added items with total price calculation
- **Navigation**: Easy navigation between Shop and Cart pages
- **Responsive Design**: Clean and simple CSS styling
- **Data Persistence**: Cart data saved using browser localStorage

## How to Run
1. Download/Clone the project folder
2. Open `index.html` in your browser
3. Or use VS Code Live Server extension for better experience
4. Click "Add to Cart" on products
5. Go to Cart page to view items and total

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage API
